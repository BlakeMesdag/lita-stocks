**lita-stocks** is a handler for [Lita](https://github.com/jimmycuadra/lita) that provides stock price information.

## Installation

Add lita-stocks to your Lita instance's Gemfile:

``` ruby
gem "lita-stocks"
```

## Usage

    Lita: stock <SYMBOL>
